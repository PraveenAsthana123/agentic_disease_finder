# Stress Detection EEG Dataset

## Overview
- **Condition**: Psychological Stress
- **Samples**: 50 (25 stressed, 25 relaxed)
- **Features**: 47 EEG features
- **Reference**: DEAP Dataset

## EEG Characteristics
- Increased beta activity
- Decreased alpha power
- Frontal asymmetry

## File
- `stress_sample_50rows.csv`: 50 samples with extracted features
