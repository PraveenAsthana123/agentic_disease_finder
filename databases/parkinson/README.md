# Parkinson's Disease EEG Dataset

## Overview
- **Disease**: Parkinson's Disease
- **Samples**: 50 (25 patients, 25 controls)
- **Features**: 47 EEG features
- **Reference**: PPMI (Parkinson's Progression Markers Initiative)

## EEG Characteristics
- Increased beta-band power (13-30 Hz)
- Reduced alpha-band coherence
- Tremor-related oscillations

## File
- `parkinson_sample_50rows.csv`: 50 samples with extracted features
