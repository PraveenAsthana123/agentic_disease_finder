# Epilepsy EEG Dataset

## Overview
- **Disease**: Epilepsy (Seizure Detection)
- **Samples**: 50 (25 seizure, 25 normal)
- **Features**: 47 EEG features
- **Reference**: CHB-MIT Scalp EEG Database

## EEG Characteristics
- Ictal spike-wave patterns
- High-frequency oscillations (HFO)
- Abnormal synchronization
