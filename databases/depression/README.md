# Depression EEG Dataset

## Overview
- **Disease**: Major Depressive Disorder (MDD)
- **Samples**: 50 (25 patients, 25 controls)
- **Features**: 47 EEG features
- **Reference**: OpenNeuro Depression Dataset

## EEG Characteristics
- Frontal alpha asymmetry
- Increased theta activity
- Reduced alpha power

## File
- `depression_sample_50rows.csv`: 50 samples with extracted features
