# Schizophrenia EEG Dataset

## Overview
- **Disease**: Schizophrenia
- **Samples**: 50 (25 patients, 25 controls)
- **Features**: 47 EEG features
- **Reference**: COBRE, RepOD Dataset

## EEG Characteristics
- Reduced P300 amplitude
- Gamma-band abnormalities
- Disrupted connectivity patterns

## File
- `schizophrenia_sample_50rows.csv`: 50 samples with extracted features
