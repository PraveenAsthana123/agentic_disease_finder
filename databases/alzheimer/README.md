# Alzheimer's Disease EEG Dataset

## Overview
- **Disease**: Alzheimer's Disease
- **Samples**: 50 (25 patients, 25 controls)
- **Features**: 47 EEG features
- **Reference**: ADNI (Alzheimer's Disease Neuroimaging Initiative)

## EEG Characteristics
- Increased slow-wave (delta/theta) activity
- Reduced alpha rhythm
- Decreased complexity measures

## File
- `alzheimer_sample_50rows.csv`: 50 samples with extracted features
