# Autism Spectrum Disorder EEG Dataset

## Overview
- **Disease**: Autism Spectrum Disorder (ASD)
- **Samples**: 50 (25 ASD, 25 controls)
- **Features**: 47 EEG features
- **Reference**: ABIDE (Autism Brain Imaging Data Exchange)

## EEG Characteristics
- Altered gamma-band activity
- Reduced functional connectivity
- Atypical neural synchronization

## File
- `autism_sample_50rows.csv`: 50 samples with extracted features
